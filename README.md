# Requirement

1. Python 3.11.2
2. requirements.txt
  
# How To Start The Game
  
```bash
$ python mainGame.py
```
### or
just change the branch to `exe`
```bash
$ git checkout exe
```
and click on the `Ooo Jaaa.exe` file
